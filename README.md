# Transcriptional Slippage Analysis

This repository contains results from a cross-species analysis of transcriptional slippage across multiple domains of life.

## Project Overview
The study analyzes homopolymer runs and their potential to cause transcriptional slippage in different organisms.

## Data Included
The repository contains results for:

- Archaea
- Eukaryotes
- Prokaryotes
- Viruses

Each species folder contains:

- `.runs.tsv` → individual homopolymer runs
- `.aa_summary.tsv` → amino acid level summary
- `.species_summary.tsv` → species level statistics
- `_SI_gene.tsv` → gene level suppression index
